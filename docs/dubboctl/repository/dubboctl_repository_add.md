## dubboctl repository add

Manage set of installed repositories.

### Synopsis

Usage:
dubboctl repository add <name> <url> [flags]

Flags:
-c, --confirm Prompt to confirm options interactively ($DUBBO_CONFIRM)
-h, --help help for add
